Find a vector that has the same direction as \(\left<2,6,-2\right>\) but has length 6.

Let us define that \( \vec{v}=\left<2,6,-2\right>\). The length of v is given by:

\(\Vert \vec{v}\Vert=\sqrt{2^{2}+6^{2}+(-2)^{2}}=\sqrt{44}=\sqrt{4}\sqrt{11}=2\sqrt{11}\)

Finding the unit vector of v.

\(\vec{u}=\dfrac{\vec{v}}{\Vert\vec{v}\Vert}= \dfrac{\left<2,6,-2\right> }{2\sqrt{11}}=\dfrac{1}{\sqrt{11}}\cdot\left<1,3,-1\right> \)

Finding the vector that has same direction as v but length 6.

\(\vec{i}=6\cdot\vec{u}=\dfrac{6}{\sqrt{11}}\cdot\left<1,3,-1\right> =\left<\dfrac{6}{\sqrt{11}},\dfrac{18}{\sqrt{11}},-\dfrac{6}{\sqrt{11}}\right> \)
